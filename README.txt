That's a GUI project where you can read the results from a tournament in the Lichess and with the results create an
excel file.


To do this project you need to get some libraries:
    openpyxl   --------  pip install openpyxl
    requests   --------  pip install requests
    selenium   --------  pip install selenium
    PyQt5      --------  pip install PyQt5
    pandas     --------  pip install pandas

After that you need the correct chromedriver for you chrome version.
